# DriveU Internal Chrome Extention.

This extension is used by On-trip team. One of the core internal team of DriveU. This extension automates following things in the day to day operations.

1. Automates the process of calling driver and customers without any API and manual work with just <b>Click of a Button</b>.
2. Automates in resolving tickets raised by the driver by just <b>Click of a Button</b>.
3. Automates the process of finding already resolved ticket without any manual work.
